# ТЗ: Игра "Азбука" — изучение русского алфавита для детей

> Кроссплатформенная игра для изучения русского алфавита детьми 3-7 лет.
> Платформы: Android, iOS, Web (PWA), macOS. Движок: Godot 4.6.

---

## 1. Мета

- Версия: 2.0
- Приоритет: high
- Статус: draft
- Дата: 2026-06-13
- Ветка: v2-roadmap

---

## 2. Цель (V2)

Добавить новые режимы и контент в игру "Азбука":
- **Карта-змейка** — навигация по буквам через серпантин с персонажем
- **Тематический фон** — хвойный лес с избушкой на курьих ножках
- **Редизайн карточки буквы** — лесной фон, зоны клика буква/картинка, разные звуки
- **Мини-игра "Найди букву в слове"** — показано слово, нужно нажать на правильную букву
- **Навигация по карточкам** — кнопки следующая/предыдущая буква
- **Голосовой ввод** — запись голоса ребёнка с возможностью воспроизведения

---

## 3. Архитектура (V2 — дополнения)

### 3.1 Новые компоненты

| Компонент | Ответственность | Связи |
|-----------|----------------|-------|
| RoadmapScreen | Карта-змейка с буквами и персонажем | → LetterDetail, AnimationPlayer |
| ForestBackground | Фоновый слой (лес, избушка, объекты) | ThemeManager, ParticleSystem |
| VoiceRecorder | Запись и воспроизведение голоса | AudioStreamMic, AudioStreamPlayer |
| GameFindInWord | Мини-игра "нажми букву в слове" | AudioManager, ProgressManager |

### 3.2 Новые data flow

```
RoadmapScreen → dot_clicked(letter_id)
  → персонаж.move_to(dot_position)
  → LetterDetail.show(letter_data)

LetterDetail (V2):
  → click_letter_zone() → AudioManager.play_letter()
  → click_picture_zone() → AudioManager.play_word()
  → click_prev/next() → анимация перехода + соседняя буква
  → click_mic() → VoiceRecorder.start_recording()
  → click_playback() → VoiceRecorder.play_recording()

GameFindInWord:
  → show_word(word)
  → on_letter_pressed(letter_index)
    → if correct: dance_animation + sparkle
    → if wrong: error_sound + shake + "Попробуй ещё"
```

## 4. Scope / Out of Scope (V2)

**In scope:**
- Экран карты-змейки с буквами (новый)
- Персонаж, перемещающийся по точкам-буквам
- Фоновый слой "хвойный лес + избушка на курьих ножках"
- Природные объекты: пенёк, камень, озеро, грибы, шишки
- Редизайн LetterDetail: лесной фон, зоны клика буква/картинка
- Кнопки навигации ← → внутри LetterDetail
- Мини-игра "Найди букву в слове" (нажми правильную букву в слове)
- Анимация успеха (буква танцует) / ошибка (грустный сигнал + "попробуй ещё")
- Голосовой ввод: запись микрофоном и воспроизведение
- Кнопка микрофона 🎤 и кнопка воспроизведения ▶️

**Out of scope:**
- Распознавание речи (STT)
- Анализ правильности произношения
- Сохранение записей на диск
- Мультиплеер
- Внутренние покупки
- Аналитика
- Другие языки

## 6. Data Models (V2 — дополнения)

```json
{
  "voice_recording": {
    "is_recording": false,
    "has_recording": false,
    "format": "PCM16",
    "sample_rate": 22050
  },
  "roadmap": {
    "current_position": 0,
    "unlocked_letters": ["а", "б", "в"]
  }
}
```

**VoiceRecording** (только runtime, не сохраняется):
| Поле | Тип | Описание |
|------|-----|----------|
| audio_data | PackedByteArray | PCM16 raw audio buffer |
| is_recording | bool | Идёт ли запись сейчас |
| has_recording | bool | Есть ли сохранённая запись |

## 10. Non-functional Requirements (V2)

- **Performance:** 60 FPS, анимации не тормозят на средних устройствах
- **Audio:** Запись микрофона — PCM16 22050Hz, только в оперативной памяти
- **Storage:** < 60MB (новые фоны + спрайты)
- **Microphone:** Запрашивать разрешение на микрофон при первом использовании
