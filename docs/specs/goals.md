# ТЗ: Игра "Азбука" — изучение русского алфавита для детей

> Кроссплатформенная игра для изучения русского алфавита детьми 3-7 лет.
> Платформы: Android, iOS, Web (PWA), macOS. Движок: Godot 4.6.

## 2. Цель

Создать красочную интерактивную игру, которая поможет детям выучить русский алфавит.

**Ключевые функции:**
- Показ букв русского алфавита (33 буквы) с яркими иллюстрациями
- Озвучка букв и слов (профессиональная дикторская запись / TTS)
- Интерактивные мини-игры для закрепления материала
- Адаптивный дизайн под экраны телефонов, планшетов и десктопов
- Работа без интернета (офлайн-режим)

**Пользовательский сценарий:**
Ребёнок открывает приложение, видит все буквы алфавита в виде карточек. Нажимает на букву — слышит её произношение и видит картинку со словом, начинающимся на эту букву. Играет в мини-игры: "Найди букву", "Собери слово", "Угадай по картинке".

## 3. Архитектура

### 3.1 Стек

- **Движок:** Godot 4.6.3 (GDScript)
- **Платформы:** Android, iOS, Web (PWA), macOS
- **2D графика:** Векторные/растровые спрайты, анимации через AnimationPlayer
- **Аудио:** OGG/Vorbis — озвучка букв и слов
- **Данные:** JSON-файлы (буквы, слова, прогресс), Resources Godot
- **Локализация:** Только русский язык (MVP)

### 3.2 Паттерны

- **Scene Tree:** Одна главная сцена, переключение через ChangeScene
- **Data-Driven:** Все данные букв/слов в JSON, загрузка через ResourceLoader
- **State Machine:** Global state (SceneManager, AudioManager, ProgressManager)
- **Signal-based:** Коммуникация через сигналы Godot

### 3.3 Компоненты

| Компонент | Ответственность | Связи |
|-----------|----------------|-------|
| AlphabetScreen | Сетка всех 33 букв | → LetterCard → LetterDetail |
| LetterDetail | Показ буквы, картинки, слова, звука | → AudioManager |
| MiniGames | Контейнер мини-игр | → GameTemplate, ScoreManager |
| AudioManager | Воспроизведение звуков букв/слов | — |
| ProgressManager | Сохранение выученных букв | → File (JSON) |

### 3.4 Data Flow

```
AlphabetScreen → on_letter_pressed(letter_id)
  → LetterDetail.show(letter_data)
    → AudioManager.play_letter(letter_id)
    → AudioManager.play_word(letter_id)

MiniGame → on_answer(correct)
  → ScoreManager.update()
  → ProgressManager.on_game_complete(letter_id)
```

## 6. Data Models

```json
{
  "alphabet": [
    {
      "id": "а",
      "letter": "А",
      "word": "Арбуз",
      "image": "res://assets/images/arbuz.png",
      "sound_letter": "res://assets/audio/a_letter.ogg",
      "sound_word": "res://assets/audio/arbuz.ogg",
      "category": "food"
    }
  ],
  "progress": {
    "completed_letters": ["а", "б", "в"],
    "games_played": 15,
    "last_played": "2026-06-13"
  }
}
```

**Сущности:**

| Поле | Тип | Описание |
|------|-----|----------|
| alphabet[].id | String | Ключ буквы (нижний регистр) |
| alphabet[].letter | String | Заглавная буква для отображения |
| alphabet[].word | String | Слово-пример |
| alphabet[].image | String | Путь к картинке |
| alphabet[].sound_letter | String | Путь к звуку буквы |
| alphabet[].sound_word | String | Путь к звуку слова |
| progress.completed_letters | String[] | Список выученных букв |
