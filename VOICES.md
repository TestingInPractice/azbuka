## 3. Подсказки и фразы в карточке буквы
| Фраза | Где используется |
|-------|-----------------|
| Найди и нажми на букву | HintLabel — подсказка в начале игры «Найди букву» (hint_find_letter.wav) |
| 🔊 Буква | Кнопка — озвучить название буквы |
| 🔊 Слово | Кнопка — озвучить слово |
| Молодец! ✨ | При правильном ответе в игре «Найди букву» |
| Попробуй ещё! | При неправильном ответе |

## 4. Интерфейс: меню
| Текст | Элемент |
|-------|---------|
| АЗБУКА | Заголовок главного меню |
| Учим русский алфавит | Подзаголовок |
| 📖 Азбука | Кнопка перехода к алфавиту |
| 🔍 Найди букву | Кнопка игры |
| 🧩 Собери слово | Кнопка игры |
| 🖼️ Угадай картинку | Кнопка игры |
| 🗺️ Карта-змейка | Кнопка карты прогресса |
| Прогресс: 0 / 33 | Счётчик прогресса |
| ⚙️ | Кнопка настроек |

## 5. Интерфейс: игра «Найди букву»
| Текст | Элемент |
|-------|---------|
| Найди букву: {буква} | Заголовок раунда (напр. «Найди букву: А») |
| {счёт} / 10 | Счёт |
| Результат: {счёт} / 10 | Финальный результат |
| Идеально! | При результате 10/10 |
| Отлично! | При результате 7-9/10 |
| Попробуй ещё! | При результате < 7/10 |
| Играть снова | Кнопка |
| ← На главную / ← Выход | Кнопка назад |

## 6. Интерфейс: игра «Угадай картинку»
| Текст | Элемент |
|-------|---------|
| Раунд {n} / {n} | Заголовок раунда |
| Счёт: {n} | Текущий счёт |
| Молодец! | При правильном ответе |
| Попробуй ещё! | При неправильном ответе |
| Игра окончена! | Финал |
| Ваш счёт: {n} / {n} | Итоговый счёт |
| Играть ещё | Кнопка |
| ← Назад | Кнопка назад |

## 7. Интерфейс: игра «Собери слово»
| Текст | Элемент |
|-------|---------|
| Раунд {n}/{n} | Заголовок раунда |
| Молодец! | При правильном ответе |
| Отлично! Все слова собраны! | Финал |
| ← Назад | Кнопка назад |

## 11. Диалоги
| Текст | Где |
|-------|-----|
| Внимание | Заголовок диалога |
| Должен быть включён хотя бы 1 режим игры. | При попытке выключить последний режим |
| Сбросить весь прогресс по азбуке? Буквы перестанут быть зелёными. | Диалог подтверждения сброса |
| Сбросить | Кнопка подтверждения |
| Отмена | Кнопка отмены |

## 12. Звуковые файлы для замены
В папке `assets/audio/` сейчас лежат сгенерированные `.wav` и `.raw` файлы.
Имена файлов (можно перезаписать своими):

| Файл | Описание |
|------|----------|
| hint_find_letter.wav | Подсказка «Найди и нажми на букву» |
| {буква}_letter.wav / {буква}_letter.raw | Название буквы (33 шт) |
| {слово}.wav | Слово (33 шт) |
